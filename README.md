## ⚡ Using the REST Gateway

- The gateway listens for incoming API requests.
- It checks each request against your authentication rules.
- If the user is not authorized, the request is rejected.
- The system tracks how many requests each user makes.
- If a user reaches the rate limit, further requests are blocked until limits reset.
- Valid requests are forwarded to your API.
